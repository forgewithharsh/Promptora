# Promptora

Promptora is a minimal, GPT-inspired AI assistant designed for clear, thoughtful, and prompt-driven conversations.  
Built as a portfolio project, it focuses on clean architecture, modern UI, and practical AI integration.

---

## Features

- Intelligent AI-powered conversations  
- Prompt-driven interaction model  
- Chat history support  
- Fast and responsive interface  
- Scalable and clean code structure  

---

## Tech Stack

- **Frontend:** React / Javascript  
- **Backend:** Node.js / Express  
- **AI:** GPT-based API integration  
- **Styling:** Tailwind CSS  

---

## Project Purpose

Promptora was built to explore:

- Real-world AI chat workflows
- Full-stack application design
- Clean UI/UX for AI products
- Production-ready project structure
